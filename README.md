# Usage
<!-- usage -->
```sh-session
$ npm install -g 3box_cli
$ 3box_cli COMMAND
running command...
$ 3box_cli (-v|--version|version)
3box_cli/0.0.0 darwin-x64 node-v10.17.0
$ 3box_cli --help [COMMAND]
USAGE
  $ 3box_cli COMMAND
...
```
